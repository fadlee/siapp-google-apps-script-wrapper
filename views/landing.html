Pertanyaan yang Sering Diajukan

Apa itu SiApp? +
SiApp adalah platform untuk memperpendek URL Google Apps Script yang panjang menjadi URL yang mudah diingat dan dapat dikustomisasi. Anda dapat mengatur nama aplikasi, ikon, dan tema untuk proyek Apps Script Anda.
Bagaimana cara menggunakan SiApp? +
Mudah! Cukup masukkan URL panjang Google Apps Script Anda, pilih nama URL pendek yang diinginkan, atur nama aplikasi dan ikon, lalu aplikasi Anda siap diakses dengan URL yang mudah diingat.
Bagaimana setting doGet() di Apps Script agar bisa jalan di iframe? +
Langkah setting Apps Script:
1. Buat fungsi doGet(e) di Apps Script
2. Return HtmlService dengan setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL)
3. Deploy sebagai web app dengan akses "Anyone"
4. Copy URL web app ke SiApp

Contoh kode:
function doGet(e) {
  return HtmlService.createTemplateFromFile('index')
    .evaluate()
    .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);
}
Apakah ada panel admin untuk mengelola aplikasi? +
Ya! Panel admin di /cp/ menyediakan dashboard lengkap untuk mengelola semua URL Apps Script Anda, melihat statistik akses, mengatur konfigurasi, dan menambahkan aplikasi baru dengan mudah.
Apakah mendukung Progressive Web App (PWA)? +
Tentu! SiApp secara otomatis menghasilkan manifest.json, service worker, dan konfigurasi PWA untuk setiap aplikasi Apps Script Anda. Aplikasi akan tampil seperti aplikasi native di perangkat mobile.
Bisakah saya mengkustomisasi tampilan aplikasi? +
Absolut! Anda dapat mengatur nama aplikasi, deskripsi, ikon kustom, warna tema, dan bahkan template HTML yang akan digunakan. Semua dapat diatur melalui panel admin yang user-friendly.
